Setup Instructions

  1. Install Ollama on your server: curl -fsSL https://ollama.com/install.sh | sh
  2. Start Ollama: ollama serve
  3. Pull a model: ollama pull llama3:8b
  4. Enter your server URL above and test connection
  5. Enable AI chord generation and save settings

Performance Note: AI generation takes 30-60 seconds per song. Best used as fallback when other sources fail.

How Multi-Source Chord Fetching Works

1
Songsterr API: Fast official API (2-3 seconds)
2
Ultimate Guitar: Large database if Songsterr fails (3-5 seconds)
3
Ollama AI: Generates chords if both fail (30-60 seconds)
4
Manual Entry: Last resort if all sources fail
